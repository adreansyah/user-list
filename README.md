### Getting Started

Clone and run application.

```sh
$ cd /{YOUR_PATH_FOLDER}/
$ git clone "https://github.com/adreansyah/user-list.git"
$ cd user-list/
```

1. Start CRA Using yarn

```sh
$ cd user-list/
$ sudo yarn install
$ sudo yarn start
```

2. Start CRA Using npm

```sh
$ cd user-list/
$ sudo npm install
$ sudo npm start
```

2. Start Testing npm

```sh
$ cd user-list/

$ sudo npm run test
```
